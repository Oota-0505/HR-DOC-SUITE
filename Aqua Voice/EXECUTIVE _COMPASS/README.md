# EXECUTIVE COMPASS - エグゼクティブ人材育成プログラム LP

## プロジェクト概要

「EXECUTIVE COMPASS」の高級感あふれるランディングページ（LP）です。**ネイビー×シルバー**のカラースキームを採用し、上場企業向けのプレミアムなデザインで、エグゼクティブ人材育成プログラムの価値を訴求します。

企業の持続的成長を担うエグゼクティブ人材の育成を支援し、人的資本経営を加速させるための人材育成プログラムを、洗練されたデザインで紹介しています。

## 🔧 カスタマイズ方法

### 色の変更

`index.html`の`<style>`タグ内の`:root`変数を編集：

```css
:root {
    --navy-dark: #0a1628;
    --navy-primary: #1a2942;
    --navy-medium: #2d3e5f;
    --navy-light: #4a5f7f;
    --silver-light: #e8ecf1;
    --silver-medium: #c5cdd6;
    --silver-dark: #8b95a5;
    --gold-accent: #c9a961;
}
```

### フォントの変更

Google Fontsのリンクと以下のクラスを編集：
- `.font-serif` - セリフフォント
- `.font-sans-alt` - サンセリフフォント

### メタリック効果の調整

`.metallic-silver`と`.metallic-gold`クラスのグラデーションを編集：

```css
.metallic-silver {
    background: linear-gradient(135deg, #e8ecf1 0%, #c5cdd6 50%, #e8ecf1 100%);
    -webkit-background-clip: text;
    -webkit-text-fill-color: transparent;
}
```
