# MEAL-demo

MEAL PREP SYSTEM のランディングページ（LP）デモです。

週末90分の作り置きで、平日の自炊が整う暮らしを提案するクラウドツールのプロモーション用ページです。

## 構成

- **index.html** — 1ページ完結のLP
- **css/style.css** — スタイル（北欧風・ウォームベージ）
- **js/main.js** — ヘッダースクロール・ハンバーガーメニュー・スムーススクロール
- **images/** — ヒーロー・サービス・フロー・会社案内などの画像

## 閲覧方法

`index.html` をブラウザで開くか、ローカルサーバーで表示してください。

```bash
# 例: Python で簡易サーバー
cd "MEAL PREP SYSTEM"
python3 -m http.server 8000
# http://localhost:8000 で表示
```
