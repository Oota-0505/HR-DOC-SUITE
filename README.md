# HR-DOC-SUITE

人事向けテンプレートサブスクリプションサービス「HR DOC SUITE」のランディングページ

## 📋 プロジェクト概要

月額制人事テンプレートサブスクリプションサービスの紹介・申し込みを目的としたランディングページ

### サービス概要

月額1,980円（税込2,178円）から、採用・評価・1on1・人事制度運用に使える"実務ドキュメントテンプレート"をまとめて利用できる、人事向けテンプレートサブスクリプションサービス

## 🎨 特徴

- **モダンなデザイン**: パララックス効果やスムーズなアニメーションを採用
- **レスポンシブ対応**: モバイル・タブレット・デスクトップに完全対応
- **スタイリッシュなUI**: カードデザインを避けた、ダイナミックなストリップレイアウト
- **パフォーマンス重視**: 軽量で高速なページ読み込み

## 🚀 セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/Oota-0505/HR-DOC-SUITE.git
cd HR-DOC-SUITE
```

2. ローカルサーバーで確認（推奨）
```bash
# Pythonの場合
python -m http.server 8000

# Node.jsの場合
npx serve
```

3. ブラウザで確認
```
http://localhost:8000
```

## 📝 セクション構成

- **HERO**: ヒーローセクション
- **VOICE**: 利用者の声（無限ティッカー）
- **SERVICE**: サービス内容（ダイナミックストリップ）
- **PROBLEM**: お悩み（カードグリッド）
- **REASON**: 選ばれる理由
- **FLOW**: ご利用の流れ
- **PLAN**: 料金プラン
- **FAQ**: よくある質問
- **CTA**: エントリー
- **COMPANY**: 会社概要
